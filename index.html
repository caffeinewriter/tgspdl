Telegram Sticker Pack Downloader

Download a sticker pack as PNGs.

Originally made because I wanted to share a sticker pack with friends that weren't on Telegram.

Instructions

  1. Sign in to https://web.telegram.org/.
  2. Open the add modal for the sticker pack you want in Telegram Web.
  3. Scroll to the bottom so that all stickers are loaded into the modal.
  4. Run this script.

Bookmarklet

Note: While I did make this with security in mind (all dependencies are included with the script itself) you should never run arbitrary code, particularly on Telegram. The best idea is to probably not use this. The second best idea is to get the code yourself, vet it, then run it locally. However, there is still a bookmarklet for your convenience.
